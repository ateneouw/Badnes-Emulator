# Badnes-Emulator
A free nintendo Emulator for Windows and Linux

Default keys are: 
* z:  B
* x:  A
* a:  SELECT
* s:  Start

This behavior can be changed selecting the joystick plugged in the menu.
I suggest using a joystick with D-PAD since it's auto configurated.
